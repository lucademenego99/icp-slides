Interactive Code Playgrounds

Ripensare l'esperienza utente delle presentazioni nelle lezioni di programmazione

Demo interattiva del progetto Interactive Code Playgrounds

Imparare a programmare non è semplice.

Specialmente se sei un principiante.

La tua università probabilmente insegna a programmare attraverso dei laboratori.

Uno dei primi passi nei laboratori è quello di far installare agli studenti un ambiente di sviluppo.

Senza un ambiente di sviluppo, non puoi iniziare a programmare.

Spesso l'installazione è piuttosto complessa.

Iniziare dovrebbe essere più semplice.

Vogliamo proporre una soluzione
che ripensa uno degli elementi fondamentali delle lezioni di programmazione.

Le umili slide in PDF.

Una tipica slide in una lezione di programmazione somiglia a questa.

Vai giù per analizzare alcuni problemi che queste slide creano, o vai a destra per continuare.

  • Il codice sulla destra è un'immagine.
  • Non puoi selezionare il testo da essa.
  • Anche se lo potessi fare, se è in una slide in PDF, molto spesso non viene selezionato correttamente.
  • Se provi a copiare ed incollare il testo, il risultato potrebbe essere, francamente, spazzatura.
  • Come insegnante, se devi cambiare qualcosa devi ricreare l'immagine.
  • A peggiorare le cose, c'è anche un problema di accessibilità:
    un'immagine è difficilmente utilizzabile per utenti che utilizzano screen reader.
  • Per finire, a volte le immagini non vengono renderizzate correttamente. Guarda questi artefatti di compressione!
  • Ma il più grande problema è questo:
  • Stai studiando, e sei curioso di sapere cosa fa questo frammento di codice?

Tough luck.

Okay, forse questo era un po' eccessivo.

Puoi scoprire cosa fa il codice, ma ci vorranno alcuni salti mortali.
Scoprilo andando giù, o continua sulla destra.

Bene, c'è del codice interessante in quella slide!
Diciamo che vuoi eseguire del codice Python...
Dovrai solo...
Aprire l'interprete Python
Aprire un editor di testo
Selezionare il codice sulla slide (se puoi farlo!)
Copialo ed incollalo nell'editor (o copialo a mano!)
Controlla se ci sono eventuali errori derivanti dal copia-incolla
Esegui il codice nell'interprete

Forza, prova ad eseguire il codice.

Hai visto cosa è successo nella sezione "Output"?

Il codice è stato eseguito direttamente nella slide!

Inoltre, puoi modificare il codice direttamente dall'editor!

Perché non provi a modificarlo, eseguire nuovamente il codice e vedere cosa accade?

Continua quando hai finito di giocarci...

Hai appena visto quello che chiamiamo un Interactive Code Playground (o ICP).
E c'è altro che possono fare...

Puoi rendere modificabili solo alcune parti di codice. Potrebbe risultare utile per eventuali esercizi guidati!

Cliccando sulle impostazioni, puoi trovare dei pulsanti per:

  • copiare il codice
  • annullare tutte le tue modifiche
  • abilitare l'utilizzo dei 'tabs' nell'editor

ICP è compatibile con altri linguaggi!

  • Javascript
  • Typescript
  • Java
  • SQL

Sulla destra, ad esempio, puoi eseguire del codice Java!

Speriamo che questi ICP possano rendere l'apprendimento della programmazione più *creativa*, e basata su *fast tinkering*.
