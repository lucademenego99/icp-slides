Modular programming: functions

  • The functions represent the main modularization tool, as they allow:
    • to reuse code (rather than duplicate it)
    • to adapt code to different situations, using parameters

The Memory Model

Let's see it step by step

function sum(a: number, b: number): number {
    let res: number;
    res = a+b;
    return res;
}

function product(b: number, a: number): number {
    let res: number = 0;
    for (let k: number = 0; k < b; k++) {
        res = sum(res, a);
    }
    return res;
}

let a: number = 2;
let b: number = 3;
let res: number = product(a,b);

console.log(a + ' * ' + b + ' = ' + res);
