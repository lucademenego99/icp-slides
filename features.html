Interactive Code Playgrounds

Main Features

Showcase of the Interactive Code Playgrounds project

Javascript / Typescript languages

  • Javascript and Typescript languages are available with syntax checking and auto completion
  • You can run the code directly in the browser, or copy it with the copy button
  • Tabs handling is not enabled by default to support people with vision impairments - enable it with the other button!

Editable fields

Try to change the type of the ID variable

  • You can allow to only edit some specific parts of the code
  • In the following example, the types of the ID field are editable
  • Even the dark theme is available!

Python

  • You can use Python, with the most famous libraries working out of the box!
  • If you would like to use a library that is not already integrated, you can install it with micropip
