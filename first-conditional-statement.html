Let's consider this simple snippet of code

int x = 0;
void draw(){
  if (x>100) {
    fill(0,255,0);
  }
  if (x>200) {
    fill(255,0,0);
  }
  ellipse(100,100,100,100);
  x=x+1;
}

What do you think it does?

Case 1

int x = 0;
void draw(){
  if (x>100) {
    fill(0,255,0);
  }
  if (x>200) {
    fill(255,0,0);
  }
  ellipse(100,100,100,100);
  x=x+1;
}

When x=0 ... 100: only lines 9-10 are executed, and a white circle is shown

Case 2

int x = 0;
void draw(){
  if (x>100) {
    fill(0,255,0);
  }
  if (x>200) {
    fill(255,0,0);
  }
  ellipse(100,100,100,100);
  x=x+1;
}

When x=101 ... 200: lines 4, 9-10 are executed, and a green circle is drawn

Case 3

int x = 0;
void draw(){
  if (x>100) {
    fill(0,255,0);
  }
  if (x>200) {
    fill(255,0,0);
  }
  ellipse(100,100,100,100);
  x=x+1;
}

When x > 200: lines 7, 9-10 are executed, and a red circle is shown
