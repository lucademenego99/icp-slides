Istruzioni condizionali

Le istruzioni condizionali ci permettono di eseguire un pezzo di codice solo se una condizione è vera.

Le istruzioni condizionali sono formate da due elementi:

  • TEST - La condizione il cui valore determina se eseguire il codice associato
    • SE la pasta sta bollendo:
  • BODY - Il blocco di codice da eseguire se la condizione è vera
    • ALLORA butta la pasta!

Istruzioni condizionali

// Sintassi da usare:
if (test) {
    statements
}
  • L'espressione test deve sempre avere valore true o false
    • NOTA: usa sempre le parentesi tonde () per il test
  • Se test è true, verrà eseguito il codice nelle parentesi graffe {}
  • Se test è false, il codice nelle graffe è ignorato.

Un esempio

Se test è true, gli statement eseguiti saranno:

  • statement 1;
  • statement 2;
  • statement 3;
  • statement 4;
  • statement 5;
if (test) {
    statement 1;
    statement 2;
}
statement 3;
statement 4;
statement 5;

Se test è false, gli statement eseguiti saranno:

  • statement 3;
  • statement 4;
  • statement 5;

Prova tu!

Esegui il codice cliccando sul tasto play e verifica quali statement vengono eseguiti

Cosa succede se al posto di true inserisci false? Provaci!

Come posso scrivere le condizioni di test?

  • Usa le espressioni relazionali
    • esse sono espressioni che possono essere valutate come true o false
    • tipicamente sono costituite da due valori confrontati tra loro tramite un operatore relazionale
  • Gli operatori relazionali sono:
    • < (minore), > (maggiore)
    • <= (minore o uguale)
    • >= (maggiore o uguale)
    • == (equivalente, operatore di uguaglianza) NON CONFONDERE CON l'ASSIGNMENT "="
    • != (non equivalente, operatore di disuguaglianza)

Indentazione

  • L'indentazione è l'inserimento di una certa quantità di spazio vuoto all'inizio di una riga di testo.
  • Questa è una convenzione usata per esprimere al meglio la struttura del codice sorgente.
    • In processing/java non è obbligatorio (ma quasi sempre utile!)
    • Ci sono dei linguaggi di programmazione dove viene richiesto (es. Python)
  • Esso rende il codice più leggibile, mette in relazione i flussi di controllo ed i costrutti utilizzati (condizioni e loop).
  • Tipicamente si utilizza il tasto di tabulazione.

Esempi dello stile

if (a < 0) {
  println("a è negativo");
  a = 0;
}
  1. La parentesi iniziale { è posizionata sulla stessa righa dell'if
  2. Il blocco di codice all'interno delle graffe è indentato
  3. La parentesi di chiusura dell'if si trova su una nuova riga, non indentata
if (a < 0)
{
  println("a è negativo");
  a = 0;
}

Questa è un'altra possibilità: la prima graffa { è posizionata su una nuova riga, non indentata
