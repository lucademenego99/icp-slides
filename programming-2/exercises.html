Exercise 1: true or false?

  1. A static method of a C class cannot be invoked without an instance (object) of C being created.
  2. A Java class defined as abstract can be used within class hierarchies with multiple inheritance.
  3. Let there be a Java class A that contains a method m(). A subclass B of A may redefine m() once by override and an arbitrary number of times by overload.
  4. Variables declared static are modifiable by private methods.
  5. The constructor of a class automatically calls the constructor of the superclass with the same parameters; if no constructor with the same signature is available in the superclass, the constructor with no parameters is called.
  6. Java has a destructor (similar to C++'s destroy()) to explicitly deallocate objects in memory.
  7. The array type int[] is a reference type, like the classes.
  8. The super keyword is used to declare very large objects.
  9. Java does not support pointers: therefore, only static memory allocation on the stack is supported, not dynamic memory allocation on the heap.
  10. An x attribute declared as public in the C class of the P1 package is visible to all other classes declared in P1, but not to classes declared in packages other than P1.
