  • riassegnare una variabile stringa

5. Immutabilità 4

Cosa è successo?

x = 'ciao'
x = x + 'mondo'
  1. Python genera la stringa 'ciao' e la assegna ad x
  2. Poi:
    • calcola il risultato di x + 'mondo', genera NUOVA stringa 'ciaomondo'
    • assegna la nuova stringa a x

5. Immutabilità 5

Per riassumere...

x = 'ciao'
x = x + 'mondo'
  1. PRIMA viene calcolato il risultato dell'espressione a destra dell'=
  2. POI il risultato viene associato alla variabile a sinistra dell'=

6. Formattazione 1

Formattare stringhe

Concatenare stringhe con un segno + come sopra è macchinoso e si presta ad errori.

Ci sono diverse soluzioni migliori, per una recensione completa ci riferiamo al sito Real Python. In particolare, guardare le più utili che sono le f-strings, disponibili per Python >= 3.6

6. Formattazione 2

Formattare con %

  • Il place-holder (segnaposto) %s verrà sostituito con una variabile
  • La variabile è definita dopo, con %x

6. Formattazione 3

Formattare con %

  • Se volessimo usare più variabili?
  • Bisogna solo aggiungere più segnaposti %s, e dopo il % mettere la lista di variabili in delle parentesi tonde
